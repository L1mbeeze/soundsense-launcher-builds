Локальный сервер

Быстрый запуск локального сервера для тестирования презентаций, медиа и интерактивных сцен. Открывается компактное окно с одной кнопкой запуска.

Для чего

  • Локальные демо
  • Тест API и медиа
  • Временные события

Особенности

  • Мини-окно управления
  • Понятный статус
  • Быстрый старт
Окно открывается поверх лаунчера.
Шаги запуска
  1. Откройте модуль Local Server.
  2. Нажмите кнопку запуска сервера.
  3. Используйте ссылку localhost для подключения.

Минимум действий

Модуль создается как отдельная легкая программа — чтобы запуск был быстрым, а управление не отвлекало от основного сценария.

Сейчас это компактная панель управления. Позже появятся настройки порта, автоматический запуск и история подключений.
